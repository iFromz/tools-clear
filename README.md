# Tools Clear cache & ram
Tools Clear cache & ram for Unix

## Getting started

### Clone this repo
 
```git
git clone git@bitbucket.org:ifromz/tools-clear.git
```
### Copy folder `tools` to root directory

```bash
mv tools-clear ~/tools
```

```bash
touch ~/tools/logs
```

## Uses
### Run command

```bash
sh ~/tools/run.sh
```
